## 🧠 What I'm Building

### 🎙 Voice-Based RAG Assistant

A conversational assistant that retrieves knowledge from a vector database and responds through **real-time voice interaction**.

The system supports **full-duplex audio communication**, allowing users to speak naturally while receiving streaming responses.

Built with:

`FastAPI • WebSockets • Vosk STT • LangChain • pgvector (HNSW) • all-MiniLM-L6-v2 embeddings`

### 📄 CareerZync — AI Resume Analysis & Career Advisor

A system that analyzes resumes to extract skills and keywords, then provides insights and answers questions based on the resume context.

The platform enables semantic search over resume content and supports conversational career exploration.

Built with:

`React • FastAPI • FAISS • LangChain • Gemini API • PyMuPDF`

### 🧾 AI-Assisted Bill Processing System

`React • FastAPI • Gemini API • Google Vision API • OpenCV • PostgreSQL`
